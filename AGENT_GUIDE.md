# 助理小幫手使用大全

這份文件會教你怎麼跟我溝通
只要輸入暗號
我就會幫你把專案問題處理好

---

## 快捷指令一覽表

直接在對話框對我說關鍵字就好

| 指令  | 功用         | 執行動作                         |
| ----- | ------------ | -------------------------------- |
| SOS   | 自主修復程式 | 我會翻閱紀錄並直接動手修好程式   |
| CHECK | 環境衝突檢查 | 跑診斷程式看父目錄有沒有東西亂掉 |
| LOG   | 查閱歷史紀錄 | 把以前解決問題的操作紀錄翻出來   |

---

## 更好用的招式

### 丟截圖最快

如果你覺得報錯訊息太長很麻煩
直接把壞掉的畫面截圖丟給我
我會自己去讀圖片裡面的內容
這樣做往往比打字還要快

### 會持續進化

我會把每次解決問題的方法都記下來
讓這個助理小幫手越來越聰明
下次遇到同樣的問題
我眨眼間就能幫你排除故障
